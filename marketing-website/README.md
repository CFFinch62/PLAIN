# PLAIN Programming Language - Marketing Website

This is the marketing website for the PLAIN programming language.

## About PLAIN

**PLAIN** = **P**rogramming **L**anguage - **A**ble, **I**ntuitive, and **N**atural

PLAIN is a programming language designed to be approachable without sacrificing competence. It prioritizes clear thinking over clever syntax, natural readability over terse notation, and honest capability over complex features.

## Running Locally

Simply open any HTML file in a web browser. No build process required.

```bash
# Open in default browser (macOS)
open index.html

# Open in default browser (Linux)
xdg-open index.html

# Open in default browser (Windows)
start index.html
```

Or use a simple HTTP server:

```bash
# Python 3
python -m http.server 8000

# Node.js (with http-server)
npx http-server

# Then visit http://localhost:8000
```
