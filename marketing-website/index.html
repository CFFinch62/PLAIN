Why PLAIN?

Clear thinking over clever syntax. Natural readability over terse notation.

📖

Natural Syntax

Code that reads like plain English. Use task, deliver, attempt, and handle instead of cryptic symbols.

🎯

Explicit Over Implicit

var declares new variables, no var assigns to existing ones. No shadowing allowed - clarity always wins.

🏷️

Type Prefixes

Optional type prefixes make code self-documenting: intAge, strName, lstItems, tblData.

🛡️

Built-in Error Handling

First-class error handling with attempt/handle/ensure blocks. No exceptions to catch.

rem: Binary choice

    display("Adult")

rem: Multiple choices
choose grade
    choice "A"

Loops

loop i from 1 to 10
    display(i)

loop item in lstItems
    process(item)

Tasks (Functions)

task CalculateTax using (amount, rate)
    var tax = amount * rate
    deliver tax

task Main()
    var result = CalculateTax(100, 0.08)
    display(result)

Key Features

Everything you need for real programming

🎯 No Shadowing

Variables can't be redeclared in inner scopes. Once a name is used, it stays consistent throughout its scope. No confusion, no bugs.

📝 String Interpolation

Natural string formatting with v"Hello {name}!" syntax. No printf formats to memorize, no concatenation chains.

🔧 Records & Types

Define custom data structures with records. Type-safe collections with list of integer and table of string to float.

📦 Module System

Organize code with packages, assemblies, and modules. Import exactly what you need with clear, explicit declarations.

🛠️ Standard Library

Comprehensive built-in functions for strings, lists, files, math, and more. Everything uses clear, descriptive names.

🐍 Python-like Blocks

Indentation-based syntax keeps code clean and readable. No curly braces to match, no semicolons to remember.

Get Started in Minutes

Install PLAIN and start writing clear, capable code today

1

Clone & Build

git clone <repository-url>
cd PLAIN
go build -o plain cmd/plain/main.go
2

Write Your First Program

rem: hello.plain
task Main()
    display("Hello, PLAIN!")
3

Run It!

./plain run hello.plain

# Output: Hello, PLAIN!
