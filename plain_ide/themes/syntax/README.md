# Syntax Themes

This directory contains 50+ bundled Geany-compatible syntax themes for PLAIN IDE.

## Bundled Themes

These themes are automatically copied to your user configuration directory (`~/.config/plain_ide/themes/syntax/`) when you first run the IDE.

## Adding Custom Themes

To add more themes:

1. Download Geany `.conf` theme files from:
   - https://github.com/geany/geany-themes
   - https://github.com/codebrainz/geany-themes

2. Copy them to: `~/.config/plain_ide/themes/syntax/`

3. Restart the IDE or reload settings

4. Select from: **View → Theme → Syntax Theme**

## Theme Format

Themes use Geany's `.conf` format with `[named_colors]` and `[named_styles]` sections.
